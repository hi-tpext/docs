#### 左侧树形导航
要点：使用`column`分割左右
```php
class User extends Controller
{
    public function index()
    {
        //****
        $left = $builder->column(1);
        $right = $builder->olumn(11);

        $treeData = $this->getTree();//以某种方式获取的数据 
        $this->assign('treeData', $treeData);

        //渲染一个自定义模板，在其中完成树形结构
        // /admin/view/user/header.html
        $left->content()->fetch('tree');
        //$left->content()->display($html);//或者通过拼接html的形式

        $table = $right->table();
        //略
        return $builder->render();
  }
}
```

```php
            $('input[name="category_id"]').val(treeNode.id);
```
