这里拿`tpextmyadmin`做介绍
```
── assets                    (资源，不需则留空，发布时复制到`/public/assets/`)
    ├── css
    ├── images
    └── js
── data                      (安装或卸载脚本，不需要安装则留空)
    ├── install.sql
    └── uninstall.sql
── src                       （源码目录）
    ├── admin                     (admin模块)
    │    ├── controller
    │    │    ├── Admin.php
    │    │    ├── Index.php
    │    │    ├── Menu.php
    │    │    ├── Operationlog.php
    │    │    ├── Permission.php
    │    │    └── Role.php
    │    ├── model
    │    │    ├── AdminMenu.php
    │    │    ├── AdminOperationLog.php
    │    │    ├── AdminPermission.php
    │    │    ├── AdminRole.php
    │    │    ├── AdminRoleMenu.php
    │    │    ├── AdminRolePermission.php
    │    │    └── AdminUser.php
    │    └── view                   (视图模板)
    │         ├──      
    │         └── ...         
    └── common
    │    ├── behavior
    │    │    ├── Auth.php
    │    │    ├── Log.php
    │    │    ├── Menu.php
    │    │    └── Setup.php
    │    ├── MinifyTool.php
    │    └── Module.php             (模块定义)
    ├── common.php
    ├── config.php                  (扩展自定义配置)
    └── helper.php　　　　　　　　　　　(扩展加载)
── composer.json
── LICENSE.txt
── README.md
```
主要有`admin`,`common`两个模块，实际中可以按需要增加其他模块，如前台`index`模块。
